# react-notes-app
Simple React app with notes using external API.

## Installation

App requires:
- react-router
- redux react-redux redux-thunk
- react-bootstrap
- webpack webpack-dev-server webpack-cli
- babel

```bash
cd react-notes-app
```

Install react-bootstrap:
```bash
npm install redux react-redux redux-thunk
```

Install webpack:
```bash
npm install --save-dev webpack webpack-dev-server webpack-cli
```

Install babel:
```bash
npm install --save-dev @babel/core @babel/preset-env babel-loader @babel/preset-react
```

Install react-redux:
```bash
npm install redux react-redux redux-thunk
```

Install react-router:
```bash
npm install react-router-dom
```

Install react-router:
```bash
npm install react-router-dom
```

To add support for class properties in Babel:
```bash
npm i --save-dev @babel/plugin-proposal-class-properties
```

## Run the application
To run application
```bash
npm run
```
